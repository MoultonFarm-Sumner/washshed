Farm Inventory App - Local Setup

This page provides everything you need to run the Farm Inventory Management System on your local machine.

Download Local Setup Package

Setup Instructions

For Windows Users:

  1. Download the zip file using the button above
  2. Extract the zip file to a folder on your computer
  3. Open the extracted folder
  4. Double-click the local-setup/start.bat file
  5. The app will automatically install dependencies and start
  6. Your default browser will open to http://localhost:3000

For Mac/Linux Users:

  1. Download the zip file using the button above
  2. Extract the zip file to a folder on your computer
  3. Open Terminal
  4. Navigate to the extracted folder: cd path/to/extracted/folder
  5. Make the start script executable: chmod +x local-setup/start.sh
  6. Run the start script: ./local-setup/start.sh
  7. Open your browser and go to http://localhost:3000

How It Works

The setup package includes:

  • A smart server script that tries to connect to the database
  • Automatic fallback to in-memory storage if database connection fails
  • Pre-configured database credentials (in the .env file)
  • Start scripts for both Windows and Mac/Linux

Note: When you run the app for the first time, it will automatically install the required dependencies. This may take a few minutes depending on your internet connection.

Troubleshooting Database Connection

If you encounter database connection errors:

  1. Check if your computer can connect to external databases (some work or school networks may block these connections)
  2. Ensure your firewall is not blocking the connection
  3. The app will automatically fall back to in-memory storage if the database connection fails
